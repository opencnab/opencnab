# OpenCnab

O OpenCnab é um leitor de Cnab, a utilização do CNAB alimenta os bancos de dados das empresas e faz com que estes estejam constantemente atualizados em relação aos recebimentos. O foco é padronizar a leitura desses arquivos por meio de uma única ferramenta.
## Instalação

# Navegue até o arquivo: ```application\config\config.php```

 Altere ```$config['base_url'] = 'http://localhost/opencnab/';``` para o seu endereço Local ou para um dominio.
## Bancos suportados

Atualmente o OpenCnab funciona com os bancos abaixo:



* Banco do Brasil

* Santander

Toda contribuição é bem vinda. Se você deseja adaptar o OpenCnab a algum outro banco, fique à vontade para explorar o código, veja como é bastante simples integrar qualquer banco à ferramenta.
